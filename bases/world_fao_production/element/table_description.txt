This dataset contains the elements description, code and measurement unit of the 3 main datasets (crop_livestock;production_indices;value_agricultural_production) of FAO's data production domain. In FAOSTAT's production domain the data is presented in the long format, therefore the elements are simply the attributes of the crop and livestock products. Eg. area_harvested and yield for the crop_livestock_production dataset.
