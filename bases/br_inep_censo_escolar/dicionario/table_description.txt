Dicionário para as variáveis que assumem multilplos valores de STRING nas tabelas do censo escolar. Todas as tabelas tem o mesmo dicionário, há uma variável justamente para identificar a tabela a qual a variável pertence.


Para saber mais acesse:
Website: https://basedosdados.org/dataset/br-inep-censo-escolar
Github: https://github.com/basedosdados/mais/tree/master/bases/br_inep_censo_escolar

Ajude a manter o projeto :)
