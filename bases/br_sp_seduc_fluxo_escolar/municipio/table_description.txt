A tabela disponibiliza dados sobre o fluxo escolar a nível de municipio. O fluxo escolar consiste em um conjunto de variáveis que indica a taxa de aprovação, reprovação e abandono para cada nível do ensino escolar: anos inicias do ensino fundamental, anos finais do ensino fundamental e anos do ensino médio.


Para saber mais acesse:

Tratamento
----------
O tratamento envolveu juntar .CSVs de todos os diferentes anos, adaptando para que houvesse perfeita harmonia entre as bases, sempre priorizando formatos mais recentes. Outra parte crucial foi a utilização do diretório de municípios do Base dos Dados para conseguir código IBGE de 7 dígitos para a base e assim facilitar possíveis junções com outras bases.
